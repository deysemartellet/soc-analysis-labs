# Endpoint Compromise Investigation

This case study simulates a multi-stage cyber attack involving PowerShell execution, external communication, privilege escalation, and persistence mechanisms.

## Objective

Investigate suspicious activity on an endpoint and determine:
- Whether the activity is malicious
- The scope and impact of the incident
- Appropriate response actions

## Scenario Overview

The investigation begins with multiple alerts:
- Suspicious PowerShell execution
- Outbound network connection
- Creation of a new privileged user account

## Structure

- `scenario.md` → Initial alert and context
- `investigation.md` → Step-by-step analysis and reasoning
- `incident-report.md` → Final report and response actions
- `evidence/` → Supporting logs and artifacts

## Skills Demonstrated

- Log analysis
- Process tree investigation
- Threat detection
- Incident response
- Security tool usage (EDR, logs)
