# Brute Force Detection Lab

This lab simulates a SOC analyst investigating a VPN authentication alert involving multiple failed login attempts.

## Objectives
- Perform alert triage
- Investigate authentication logs
- Identify brute force patterns
- Assess impact
- Produce an incident report

## Scenario
A SIEM alert reports multiple failed login attempts targeting VPN accounts.

The analyst must determine:
- whether the activity is malicious
- whether any accounts were compromised
- what actions should be taken.
